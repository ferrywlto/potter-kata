# Potter Kata Book Store
This program will ask for inputting the book id (1 to 5) one by one.
When receiving the 'exit' command the program will calculate and display the total cost of the books in the cart. 

## Getting started

### Run the program
```bash
cd src
dotnet run
```

### Run the unit tests
```bash
cd tests
dotnet test
```
